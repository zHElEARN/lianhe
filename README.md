# Web聊天

使用 Next.js 和 React 构建的聊天前端网页。

## 功能特色

- 💬 实时聊天界面
- 🎨 现代化的 UI 设计
- 🌙 深色/浅色主题切换

## 技术栈

- **前端框架**: Next.js 15.3.4
- **UI 组件**: React 19 + Radix UI
- **样式**: Tailwind CSS 4
- **图标**: Lucide React
- **包管理**: pnpm

## 安装依赖

启用pnpm：

```bash
corepack enable pnpm
```

然后安装项目依赖：

```bash
pnpm install
```

## 启动项目

### 开发环境启动

启动开发服务器：

```bash
pnpm run dev
```


### 生产环境启动

如果需要启动生产环境服务：

```bash
pnpm run start
```


## 编译项目

### 构建生产版本

构建优化后的生产版本：

```bash
pnpm run build
```
